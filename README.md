# 🚀 Automation Testing MyKujang - Cypress

Repository ini berisi automation testing untuk aplikasi **MyKujang** menggunakan **Cypress** dengan pendekatan:

- ✅ Page Object Model (POM)
- ✅ Data Driven Testing (JSON & Excel)
- ✅ File Upload Testing
- ✅ Intercept API
- ✅ Assertion Validation

Project ini dibuat untuk menguji beberapa modul utama dalam sistem MyKujang secara otomatis.

---

# 🛠 Tools & Technology

- Cypress
- JavaScript
- Node.js
- JSON Fixture
- Excel Data Looping
- Page Object Model (POM)
